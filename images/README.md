# 專案縮圖

把每個專案的截圖放進這個資料夾，建議檔名（小寫）：
`classswift.jpg` `dna-brush.jpg` `viewsonic-campus.jpg` `viewar.jpg`
`universe.jpg` `huang-yi.jpg` `weapolution.jpg` `atlantis.jpg`

放好後，到 `index.html` 把對應卡片的 `<div class="...-thumb ph" data-label="...">`
換成 `<img src="images/檔名.jpg" alt="專案名稱 預覽" class="...-thumb">`（移除 `ph` 佔位 class）。

建議尺寸：一般卡片約 800×500、主打約 1000×600，JPG/WebP 壓縮過。
